My Projects

Multiclass Image Classification using Transfer Learning

  • Developed a 3-class classification CNN model using transfer learning.
  • Used trained weights from ResNet50, VGG16 to model the CNN to classify car brands.
  • Reported the accuracy variation of the two models.
  • Tech Stack: Tensorflow2, Transfer Learning

Indoor Navigation for Retail Store

  • An app for searching, listing and navigation to the aisles for a given list of products in a
    retail store.
  • Webscraped the store website to create a sqlite database of products, appended a new
    feature aisle number.
  • Developed a REST API in Flask to: Search for the location details of the product, provide
    the list of products to get the respective aisle numbers, get an optimal path for the shopping
    trip for the list of products.
  • The endpoint tasks were handled by CRUD queries on the database.
  • The API was consumed in an Android App using Retrofit.
  • Tech Stack: Python, urllib, BeautifulSoup, SQLITE, Flask, Heroku

Early Sepsis Detector

  • This project was made during SIH 2020 and was recommended for finals.
  • Implemented ensemble based classifier models on the highly imbalanced physiological dataset.
  • Deployed the trained model as Rest API using Flask on Heroku to be consumed by an Android frontend.
  • A frontend application was developed in Android to consume this API, which has interface,
    to enter the vital signs and trigger email notification to the attending doctor in case of
    anomaly or sepsis positive.
  • The app maintains all records of patients hospital wise in a firebase database.
  • Tech Stack: sklearn, Tensorflow2, XgBoost, imblearn, Flask, Heroku, firebase

Sentiment analysis on Twitter stream and Stock Price Prediction

  • Analysed trend in Market sentiment from tweets related to companies in observation.
  • Used different NLP algorithms like BOW, tfidf on the tweets, used textblob to classify the
    polarity of the tweets.
  • Used the sentiment output as indicator, developed models on stock data to predict the trend and stock movements.
  • Tech Stack: NLP, Time Series, Python

Heart Disease Classification using machine Learning

  • Applied statistical tests to check confidence of prediction.

  • Developed various Ml Models, and implemented their ensembles to get maximum precision
  • Tech Stack: sklearn, Tensorflow, ensemble modelling
